kkabi · Firebase 설정이 필요합니다

이 앱을 사용하려면 무료 Firebase 프로젝트를 한 번만 셋업해야 해요. 10분이면 끝납니다.

  1. Firebase 콘솔에 접속해 구글 로그인
  2. "프로젝트 만들기" → 이름 아무거나 → 만들기
  3. 왼쪽 메뉴 "Build" → Realtime Database → "데이터베이스 만들기" → 위치 아시아권 → 보안 규칙 "테스트 모드"
  4. 왼쪽 위 톱니바퀴 ⚙️ → "프로젝트 설정" → "내 앱"에서 </> (웹) 아이콘 클릭
  5. 앱 이름 입력 → "앱 등록" → firebaseConfig 객체가 나타나면 통째로 복사
  6. 이 HTML 파일을 텍스트 에디터로 열어 FIREBASE_CONFIG 부분을 복사한 객체로 교체 후 저장
